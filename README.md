# s3uri-opener
Navigate to the AWS web console corresponding to your s3uri on web pages

# Example

click this text

```
s3://my-bucket/dir1/dir2/key=k1/key2=k2
```

to go to
```
Amazon S3 > Buckets > my-bucket > dir1/ > dir2/ >  key=k1/ > key2=k2
```

and
```
s3://abc/def/
```
for
```
Amazon S3 > Buckets > abc > def/
```
